MyCriAtomMixer
==============
ADX2LEに簡易ミキサー表示を追加するEditor拡張です。
http://youtu.be/G8qsCBLrI3Q

■使い方<br>

●セットアップ<br>
ADX2LE for Unity を使用しているプロジェクトの<br>
Assets<br>
└Editor<br>
　└CriWare<br>
　　└CriAtom<br>
　　  └MyCriAtomMixer.cs ←ここにコピー<br>

●ミキサー表示  
UnityEditorで、<br>
CRI>My>Open Cri Atom Mixer ...のメニューを開く<br>
